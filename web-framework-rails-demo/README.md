# Rails Web 框架演示

Rails 全栈 Web 框架的基础演示项目。

## 特性

- 完整的 MVC 架构
- 内置 ORM (Active Record)
- 自动路由生成
- 内置测试框架
- 丰富的生成器

## 快速开始

### 安装依赖
```bash
bundle install
```

### 运行应用
```bash
rails server
```

访问 [http://localhost:3000](http://localhost:3000) 查看效果。

## 项目结构

```
├── app/              # 应用代码
│   ├── controllers/  # 控制器
│   ├── models/       # 模型
│   └── views/        # 视图
├── config/           # 配置文件
├── db/               # 数据库文件
└── Gemfile           # 依赖管理
```
